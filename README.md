## 🧠 Sobre mim

Sou profissional de Dados com formação em Economia, atuando na interseção entre **negócio e engenharia de dados**.

Trabalho estruturando pipelines analíticos, modelagem dimensional e padronização de métricas para garantir uma fonte única de verdade para tomada de decisão.

Minha atuação envolve:

- Modelagem de dados (Star Schema)
- Construção de camadas analíticas (Raw → Staging → Marts)
- Padronização de KPIs e regras de negócio
- Estruturação de camada semântica (Power BI & Looker)
- Garantia de qualidade e confiabilidade dos dados

Meu foco é transformar dados brutos em ativos estratégicos escaláveis.
